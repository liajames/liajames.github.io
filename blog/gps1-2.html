Pair programming is done to complete an objective between two people that allows for collaborative learning from each other's practices.

Pair programming also prepares us for being developers in the real world, as pairing is a common practice to complete tasks. It is also a great way to learn from each other.

The navigator acts as the guide in a coding challenge to help the driver complete a task. The driver is in control of the keyboard and is moving through the task.
